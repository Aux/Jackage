# Jackage
A multi-platform desktop app for creating, maintaining, and sharing custom entries for various Jackbox games 

## Editor Coverage
Because Jackage is primarily a text-based editor, some minigames aren't ever going to be compatible with the program. The following is a list of the few that are potentially viable candidates for implementation.
